Public/Private fields in JavaScript

Public

The members of an object are all public members.

                      var event = {
                          name: 'Mariinsky Theatre Orchestra with Gergiev',
                          date: '22.05.2017',
                          time: '19.30',
                          place: 'Bulgaria Hall',
                      }

                      console.log(`Event name: ${event.name}`);

Create Public members in Constructor


                        var Event = function(name, date, time) {
                            this.name   = name;
                            this.date   = date;
                            this.time   = time;
                        }
                        var event1 = new Event(
                            'Mariinsky Theatre Orchestra with Gergiev',
                            '22.05.2017',
                            '19.30'
                        );
                        console.log(`Event name: ${event1.name}`);

Create Public members in Prototype

Usually used for methods members:

			Event.prototype.setReminder = function(minutesBefore) {
				alert('Reminder set for '+  minutesBefore +' minutes before start!');
			};

Private

Can be accessed only by the object itself
Ordinary vars and parameters of the constructor becomes the private members.

Private methods example


			var Event = function(id) {
				// private data
				var createdOn = new Date();
				//private method
				var dateToString = function(date, sep){
					var mm = date.getMonth() + 1; // getMonth() is zero-based
					var dd = date.getDate();

					return [
					        (dd>9 ? '' : '0') + dd,
					        (mm>9 ? '' : '0') + mm,
					        date.getFullYear(),
					].join(sep);
				}
				console.log(`Onject created on: ${dateToString(createdOn, '.')}`);
			}

Private methods cannot be called by public methods.

To make private methods useful, we need to introduce a privileged method.

Privileged methods

Can access private members
Can be invoked by public methods
Its not possible to alter from outside a privileged method (though, can be deleted)
Privileged methods are assigned with this within the constructor.

Privileged methods example


			var Event = function() {
				//private members
				var dateToString = function(date, sep){
					var mm = date.getMonth() + 1; // getMonth() is zero-based
					var dd = date.getDate();

					return [
					        (dd>9 ? '' : '0') + dd,
					        (mm>9 ? '' : '0') + mm,
					        date.getFullYear(),
					].join(sep);
				}
				var timeToString = function(date, sep){
					 var seconds = date.getSeconds();
					 seconds = seconds>9? seconds : '0'+seconds;
					return date.getHours()+sep+date.getMinutes()+sep+seconds;
				}
				var createdDateTime = new Date();

				//privileged members
				this.createdDate = function(){
					return dateToString(createdDateTime, '.');
				};
				this.createdTime = function(){
					return timeToString(createdDateTime, ':');
				};
			}

			var event1 = new Event;
			console.log(event1.createdDate());
			console.log(event1.createdTime());
